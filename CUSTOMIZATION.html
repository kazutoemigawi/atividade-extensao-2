📚 Guia Completo de Customização

1. Estrutura do Projeto

  • index.html — Página inicial
  • musculos.html — Treinos por músculo
  • nutricao.html — Nutrição por biotipo
  • styles.css — Estilos principais (CSS responsivo)
  • styles-customization.css — Exemplos de temas e animações
  • scripts.js — Menu móvel e interações
  • README.md — Documentação principal

2. Personalizando Cores

Edite as variáveis CSS em styles.css:

:root{ --primary: #0c8659; /* Verde princial */ --primary-dark: #066b46; /* Verde escuro */ --secondary: #ffa500; /* Laranja destaque */ --bg: #f0f4f8; /* Fundo da página */ --card-bg: #fff; /* Fundo dos cards */ --text: #1a1a1a; /* Texto principal */ --text-light: #666; /* Texto secundário */ --border: #e0e6ed; /* Bordas */ }
💡 Dica: Use um gerador de paleta de cores para encontrar combinações harmônicas.

3. Adicionando Exercícios

Edite musculos.html e copie a estrutura de um card:

<article class="muscle-card"> <div class="icon">🔥</div> <h2>Nome do Exercício</h2> <ul> <li>Exercício 1: sets x reps</li> </ul> <h3>Progressão</h3> <ol> <li>Semana 1: descrição</li> </ol> <p>Dica: ...</p> </article>

4. Aplicando Temas

Inclua o arquivo de customização no <head>:

<link rel="stylesheet" href="styles-customization.css">

Depois, adicione a classe ao <html>:

<html lang="pt-BR" class="theme-blue">
Temas disponíveis:
  • theme-blue — Azul e verde
  • theme-purple — Roxo e rosa
  • Padrão (verde e laranja)

5. Adicionando Imagens Reais

Para melhorar ainda mais, adicione imagens em cada card:

<article class="muscle-card"> <img src="peito.jpg" alt="Exercício de peito" style="width:100%; height:150px; object-fit:cover; border-radius:8px; margin-bottom:12px;"> <!-- resto do conteúdo --> </article>
📸 Fontes de imagens livres:
  • Unsplash — Imagens de qualidade
  • Pexels — Foco em esportes
  • Pixabay — Muitas opções

7. Deploy (Publicar Online)

Opções gratuitas:
  • GitHub Pages — Hospede diretamente do GitHub
  • Netlify — Drag & drop, muito fácil
  • Vercel — Otimizado para web estática
  • Firebase Hosting — Google Cloud gratuito

8. SEO e Acessibilidade

  • Adicione <meta name="description"> em cada página
  • Use alt="" descritivo em imagens
  • Mantenha contraste de cores adequado (WCAG AA)
  • Teste em leitores de tela: NVDA

9. Performance

  • Comprima imagens: TinyPNG
  • Minimize CSS/JS (em produção)
  • Use lazy loading para imagens
  • Teste com PageSpeed Insights
