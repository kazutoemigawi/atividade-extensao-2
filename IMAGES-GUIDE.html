2. Como Substituir Imagens

Para trocar uma imagem, edite a URL no atributo src:

<img src="URL_AQUI" alt="descrição" class="card-image">

Fontes Recomendadas (Livres):

  • Unsplash - unsplash.com (URLs de alta qualidade)
  • Pexels - pexels.com (foco em esportes)
  • Pixabay - pixabay.com (muitas opções)
  • Freepik - freepik.com (ilustrações + fotos)
💡 Dica: Procure por termos como "workout", "gym", "fitness", "nutrition" nas plataformas acima.

3. Otimizar Imagens para Web

Compressão:

  • TinyPNG - Comprimi automaticamente
  • ImageCompressor - Sem criar conta
  • ImageOptim (Mac) - Ferramenta local gratuita

Conversão para WebP:

WebP reduz tamanho em ~30% vs JPG. Use um conversor online ou ImageMagick:

cwebp imagem.jpg -o imagem.webp -q 80

Depois use no HTML com fallback:

<picture> <source srcset="imagem.webp" type="image/webp"> <img src="imagem.jpg" alt="descrição"> </picture>

4. Lazy Loading (Performance)

Carregue imagens sob demanda para melhorar performance:

<img src="imagem.jpg" alt="descrição" loading="lazy" class="card-image">
✅ Suporte: Chrome 76+, Firefox 75+, Safari 15.1+

5. Srcset para Imagens Responsivas

Serve diferentes resoluções conforme o dispositivo:

<img src="imagem.jpg" srcset="imagem-small.jpg 600w, imagem-medium.jpg 1000w, imagem-large.jpg 1400w" sizes="(max-width: 600px) 100vw, (max-width: 1000px) 50vw, 33vw" alt="descrição">

6. CSS para Imagens

Adicionar filtros:

.card-image { filter: brightness(1.1) contrast(1.1); transition: filter 0.3s; } .muscle-card:hover .card-image { filter: brightness(1.2) contrast(1.15); }

Aspect Ratio (manter proporção):

.card-image { aspect-ratio: 16 / 9; object-fit: cover; }

7. Alt Text (Acessibilidade)

Sempre preencha alt="" com descrição clara:

✅ <img alt="Homem fazendo press de banco"> ❌ <img alt="imagem"> ❌ <img alt="">

8. Performance Tips

  • Resize: Use dimensões corretas (não 4000px se exibe 400px)
  • Format: WebP > JPG > PNG
  • Quality: 75-85% é bom balanço qualidade/tamanho
  • CDN: Considere Cloudinary, Imgix para otimização automática
📊 Teste: Use Google PageSpeed Insights para medir impacto.

9. Exemplo Completo
