DICAS IMPORTANTES:

1. BUSCA FUNCIONA ASSIM:
   - Digite "press" → mostra "Press de banco", "Desenvolvimento"
   - Digite "perna" → mostra "Pernas" e "Panturrilhas"
   - Digite "curl" → mostra "Curl com barra"
   - ESC → limpa a busca

2. PARA CUSTOMIZAR CORES:
   Abra styles.css e procure por :root
   Mude --primary e --secondary

3. PARA ADICIONAR MAIS EXERCÍCIOS:
   Copie um card do musculos.html e customize

4. PARA TROCAR IMAGENS:
   Procure por <img src="https://...">
   Cole uma URL nova (Unsplash, Pexels)

5. PARA DEPLOY:
   - Upload para GitHub
   - Ativa GitHub Pages (Settings → Pages)
   - Seu site estará online!

RECURSOS ADICIONAIS:

✨ JavaScript Avançado (Opcional):
   Se quiser recursos extras (print, histórico, highlight):
   Adicione ao final do <body> em musculos.html:
   <script src="scripts-advanced.js"></script>

🎨 Temas Alternativos:
   Veja styles-customization.css para temas blue/purple

📱 Responsivo:
   - Desktop: Menu horizontal
   - Tablet: Menu normal
   - Mobile: Menu hamburger ☰

SUPORTE/AJUDA:

❓ Busca não funciona?
   → Verifique se scripts.js está carregado
   → Abra DevTools (F12) e procure por erros

❓ Imagens não aparecem?
   → Verifique a URL (deve começar com https://)
   → Teste em outro navegador

❓ Menu não abre no mobile?
   → Limpe cache (Ctrl+Shift+R)
   → Teste em outro navegador
