📱 Resumo do Projeto: Treino & Nutrição

🎯 Objetivo Completo

Site responsivo e moderno com dicas de treino por músculo e nutrição adaptada a diferentes biotipos corporais, com busca e filtros interativos.

✅ Funcionalidades Implementadas

1️⃣ Estrutura Base

  • Página inicial com call-to-action (index.html)
  • Página de treinos por músculo com 6 grupos (musculos.html)
  • Página de nutrição por biotipo (3 tipos) (nutricao.html)
  • Menu responsivo com hamburger em mobile
  • Footer com navegação

2️⃣ Design e Responsividade

  • CSS Grid responsivo (3 colunas → 2 → 1)
  • Paleta de cores profissional (verde #0c8659 + laranja #ffa500)
  • Cards com efeitos hover e sombras
  • Breakpoints: Desktop (1200px+), Tablet (768-1200px), Mobile (<768px)
  • Tipografia hierárquica clara
  • Animações suaves (fade-in, transform)

3️⃣ Imagens Integradas

  • 11 imagens de alta qualidade (Unsplash)
  • Otimizadas com dimensões corretas (400x200/250px)
  • object-fit: cover para consistência
  • Alt text descritivo para acessibilidade

4️⃣ Busca e Filtros (Novo!)

  • Busca em tempo real por nome, exercício ou dica
  • Contador de resultados dinâmico
  • Atalho ESC para limpar busca
  • Feedback visual (✅ / ❌)
  • Animação ao carregar resultados

🚀 Como Usar a Busca

Busca Básica (Ativada)

Acesse: musculos.html
Procure: Campo "🔍 Buscar exercício"
Digite: ex. "press", "agachamento", "perna"
Resultado: Cards filtrados em tempo real com contador

Buscas Avançadas (Opcional)

Inclua em musculos.html:
<script src="scripts-advanced.js"></script>

Novos recursos:
  • Destaque de palavras encontradas
  • Histórico de buscas recentes (localStorage)
  • Botão de impressão dos resultados
  • Atalhos: Ctrl+F (busca), Ctrl+P (imprimir)

🎨 Personalização Rápida

Trocar Cores

Edite em styles.css no :root:
--primary: #0c8659; → Mude a cor verde
--secondary: #ffa500; → Mude a cor laranja

Trocar Imagens

Nos HTMLs, procure por <img src="https://...">
Cole uma URL nova de imagem (Unsplash, Pexels, etc.)

Adicionar Exercícios

Copie um card do musculos.html e customize
Use o mesmo formato de estrutura HTML
