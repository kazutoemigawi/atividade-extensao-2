# 💪 Site de Dicas de Treino e Nutrição

Um site responsivo e moderno com orientações práticas sobre treinos por músculo e nutrição para diferentes biotipos corporais.

## 📋 Conteúdo

### Páginas
- **Página inicial** (`index.html`) - Apresentação e links rápidos
- **Treinos por Músculo** (`musculos.html`) - 6 grupos musculares com progressões de 4 semanas
- **Nutrição por Biotipo** (`nutricao.html`) - Recomendações para ectomorfo, mesomorfo e endomorfo

## 🎯 Características

✅ Design responsivo (mobile-first)  
✅ Paleta de cores moderna e acessível  
✅ Menu hambúrguer para dispositivos pequenos  
✅ Ícones emojis para melhor visualização  
✅ Cards com efeitos hover  
✅ Hierarquia visual clara  
✅ Tipografia legível  

## 🚀 Como usar

1. Abra `index.html` no navegador
2. Navegue entre as seções usando o menu
3. Customize cores, conteúdo e imagens conforme necessário

## 🎨 Personalização

### Cores (em `styles.css`)
Edite as variáveis CSS no `:root`:
```css
--primary: #0c8659;        /* Verde principal */
--secondary: #ffa500;      /* Laranja destaque */
--bg: #f0f4f8;            /* Fundo */
```

### Conteúdo
Edite diretamente nos arquivos `.html` para adicionar mais exercícios, biotipo ou dicas.

### Imagens
Adicione imagens em tags `<img>` ou use mais emojis conforme preferência.
