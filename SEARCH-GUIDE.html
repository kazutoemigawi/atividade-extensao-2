🔍 Guia de Busca e Filtros de Exercícios

1. Como Usar a Busca

Na página Treinos por Músculo, você encontrará um campo de busca no topo:

Campo: 🔍 Buscar exercício (ex: press, agachamento, remada...)

Exemplos de buscas:

  • press — Encontra "Press de banco", "Desenvolvimento militar"
  • agachamento — Encontra "Agachamento livre"
  • remada — Encontra "Remada curvada"
  • perna — Encontra toda a seção de pernas
  • curl — Encontra "Curl com barra"
  • progressão — Busca na seção de dicas
💡 Dica: A busca é insensível a maiúsculas, então "PRESS" = "press" = "Press".

2. O que a Busca Encontra

A busca localiza em 3 áreas de cada card:

  • Nome do músculo: "Peito", "Costas", "Pernas", etc.
  • Exercícios e variações: "Press de banco", "Fly", "Curl", etc.
  • Dicas práticas: "amplitude", "controle", "postura", etc.

3. Atalhos de Teclado

  • ESC — Limpa a busca e mostra todos os exercícios novamente
  • Clique no campo — Ativa o modo de busca com foco automático

4. Feedback Visual

Ao buscar, você verá um contador de resultados:

✅ 2 exercícios encontrados — Sucesso! Múltiplos resultados
✅ 1 exercício encontrado — Sucesso! Um único resultado
❌ Nenhum exercício encontrado — Tente outros termos

5. Como Funciona o Filtro (Código)

O JavaScript filtra em tempo real:

const query = searchInput.value.toLowerCase(); const cards = exercisesGrid.querySelectorAll('.muscle-card'); cards.forEach(card => { const title = card.querySelector('h2').textContent.toLowerCase(); const exercises = card.querySelector('ul').textContent.toLowerCase(); const tip = card.querySelector('p').textContent.toLowerCase(); if(title.includes(query) || exercises.includes(query) || tip.includes(query)) { card.style.display = 'block'; } else { card.style.display = 'none'; } });

7. Integração com Nutrição

Você pode adicionar a mesma busca em nutricao.html:

<input type="text" id="search-input" class="search-box" placeholder="🔍 Buscar biotipo (ex: ectomorfo, caloria...)">
Exemplo: Buscar "caloria" encontraria info de calorias para cada biotipo.

9. Exemplo de Uso Prático

Cenário: Você quer treinar apenas costas.

  1. Clique no campo de busca
  2. Digite "costas" ou "remada"
  3. Veja apenas exercícios de costas
  4. Copie a progressão no seu caderno de treino
  5. Pressione ESC para voltar à lista completa

10. Troubleshooting

Problema Solução
Busca não funciona Verifique se scripts.js está carregado
Campo de busca não aparece Abra DevTools (F12) e procure por erros
Resultados "quebrados" Verifique se o HTML tem id="exercises-grid"
